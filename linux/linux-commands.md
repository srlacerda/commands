# linux-commands
some linux commands

## commands
commands


1 - discover your user (who am i)
```
whoami
```

2 - show directorys
```
ls
```

3 - show directorys and some information
```
ls -l
```

4 - show directorys, hidden directorys and some information
```
ls -la
```

5 - current directory
```
cd .
```

6 - previous directory
```
cd ..
```

7 - show a manual of the command
```
man ls
```

touch file.txt
